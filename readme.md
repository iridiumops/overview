# Iridium Overview

Simple yet powerful EVE Online overview pack for everyone, every situation and all play-styles. Designed with multiple window layouts in mind, modular and easily configurable.

**Features:**
- 12 preconfigured tabs
- 38 overview filters, see [full filter list](parts/readme.md)
- 3 bracket filters
- 3 dscan filters
- standard color-coding
- readable ship labels

## Just try it!

### Install from ingame channel

![Install guide](https://i.imgur.com/zGVtzh3.png)

- Join ingame channel `Iridium Overview` 
- Reset your overview settings and close the window. [How to reset](imgur.com/a/2EtPh0J) (recommended)
- Click all install links
- Install any modifications to the base pack (optional)
- Log out and log back in (recommended)

### Install via YAML file
- Get the latest version from [Releases](https://github.com/iridiumops/overview/releases)
- Download the YAML file(s) to `%USERPROFILE%/Documents/EVE/Overview/`
- Ingame, open overview settings
- Reset your overview settings (recommended)
- On `Misc` tab click `Import Settings` button
- Select the downloaded file in the list
- Tick all filters and click `Import` button
- Close the settings window. Then log out and log back in (recommended)
